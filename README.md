# AI-Real-Estate-Advisor

## Overview

AI Real Estate Advisor is an ongoing project focused on learning and applying machine learning to predict house prices. This project will gradually evolve into a generative AI-based system that provides smart real estate insights.

## Objective

* Learn Python for machine learning
* Understand core ML concepts
* Build a house price prediction model
* Extend the project into a GenAI-powered advisor


## Current Status

* Project started
* Learning Python libraries (NumPy, Pandas)
* Dataset exploration in progress
