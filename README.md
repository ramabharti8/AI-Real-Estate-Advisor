# AI Real Estate Advisor

## Overview  
AI Real Estate Advisor is an end-to-end Machine Learning and Generative AI project designed to predict property prices and provide intelligent real estate insights.

The project begins with building a strong foundation in regression-based price prediction and will evolve into a Generative AI-powered system that delivers smart recommendations, explanations, and investment guidance.

## Problem Statement  
Estimating the fair price of a property is challenging due to multiple influencing factors such as location, size, amenities, and market trends.

Buyers and investors often lack:
- Accurate price estimation  
- Data-driven insights  
- Clear investment guidance  

This project aims to address these challenges using Machine Learning and Generative AI.

## Objectives  
- Learn and apply Python for Machine Learning  
- Understand end-to-end ML workflow
